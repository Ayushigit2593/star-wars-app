# Getting Started with the application

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

- Run npm install to install all dependencies for the project.
- Run npm start to run the project on browser. It will run the app in the development mode.

## Functionality completed

- Showing three buttons on the landing page of the app.(People, Planet, Movie)
- On click of the button, new route will render the corresponding list of data.
- On click of any item of the list, details screen is displayed.

### Techniques used

- Redux toolkit for state management.
- Lazy loading for changing routes.
- Material UI integration for styling components.
- Created project with typescript template.
